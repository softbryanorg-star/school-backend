# School Website Starter

## Admin Login
<!-- - URL: /login -->
- Secure, only authorized users

## Pages
- Create, edit, delete pages
- WYSIWYG editor
- SEO fields: metaTitle & metaDescription

## News
- Create, edit, delete announcements
- SEO fields available

## Gallery
- Upload school photos via Cloudinary
- Delete images

## Contact Messages
- View parent/student messages
- Mark as read

## School Info
- Update school name, address, phone, logo
- Appears site-wide

## Logout
- Click logout to secure account

## Deployment
- Use `.env` for credentials
- Run backend: `node server.js`
- Run frontend: `npm run dev` (Vite) or `npm start` (CRA)
